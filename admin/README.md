# CYY家具後台管理系統

## 🎯 系統概述

完整的後台管理系統，支持分級權限管理、身份確認、商品管理和實時同步到主頁功能。

## 🔑 登錄憑證

### 超級管理員

- **密碼：** `admin123`
- **權限：** 全部功能（產品、分類、訂單、用戶、統計、設定）

### 商品管理員
- **用戶名：** `CYY302630`
- **密碼：** `1`
- **權限：** 產品管理、分類管理、儀表板

## ✨ 核心功能

### 1. 權限管理系統
- ✅ **分級權限控制**：根據管理員角色顯示不同功能模塊
- ✅ **身份確認機制**：重要操作前需要密碼確認
- ✅ **菜單動態隱藏**：無權限的菜單項自動隱藏

### 2. 產品管理功能
- ✅ **完整CRUD操作**：新增、編輯、刪除、狀態切換
- ✅ **批量操作**：支持批量啟用、停用、刪除
- ✅ **搜索篩選**：按名稱、分類、狀態篩選
- ✅ **圖片管理**：支持圖片上傳和預覽
- ✅ **庫存管理**：實時庫存數量管理
- ✅ **統計信息**：產品總數、啟用數、庫存預警

### 3. 實時同步功能
- ✅ **主頁同步**：後台產品變更實時同步到主頁
- ✅ **狀態同步**：只有啟用的產品會顯示在主頁
- ✅ **數據持久化**：使用localStorage保存數據

### 4. 用戶體驗
- ✅ **響應式設計**：支持桌面和移動設備
- ✅ **現代化UI**：Louis Vuitton風格設計
- ✅ **操作反饋**：所有操作都有明確的成功/失敗提示
- ✅ **數據驗證**：完整的前端數據驗證

## 🚀 快速開始

1. **訪問登錄頁面**
   ```
   打開：admin/login.html
   ```

2. **使用超級管理員登錄**
   ```
   用戶名：admin032630
   密碼：admin123
   ```

3. **測試產品管理功能**
   - 點擊側邊欄「產品管理」
   - 嘗試新增產品
   - 測試編輯和刪除功能
   - 使用搜索和篩選功能

## 🔧 技術架構

### 前端技術
- **HTML5 + CSS3** - 現代化標記和樣式
- **JavaScript ES6** - 原生JavaScript，無框架依賴
- **Bootstrap 5.3** - 響應式UI框架
- **Font Awesome 6** - 圖標庫

### 數據存儲
- **localStorage** - 後台管理數據持久化
- **sessionStorage** - 臨時會話數據
- **JSON格式** - 結構化數據存儲

### 權限設計
```javascript
adminRoles = {
    'admin032630': { 
        level: 'super_admin', 
        name: '超級管理員', 
        permissions: ['all'] 
    },
    'CYY302630': { 
        level: 'product_manager', 
        name: '商品管理員', 
        permissions: ['products', 'categories', 'dashboard'] 
    }
}
```
